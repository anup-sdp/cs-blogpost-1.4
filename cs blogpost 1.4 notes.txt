part 10:
pip install "pwdlib[argon2]" # (recommended) securely hash user passwords (passlib is older)
pip install pyjwt # provides tools to encode/decode JWT tokens (e.g., creating tokens on login, verifying tokens in protected routes)(alternative: python-jose)
pip install pydantic-settings (... + supports .env file)
