using pip vs uv:

 The Manual Way (Old)            The uv Way (Modern)                      
------------------------------------------------------------------------
 python -m venv .venv            uv venv (or just let uv run create it)   
 source .venv/bin/activate       Optional. Just use uv run <command>      
 pip install fastapi             uv add fastapi                           
 pip freeze > requirements.txt   Automatically handled in uv.lock         
------------------------------------------------------------------------
run app: uv run uvicorn main:app
