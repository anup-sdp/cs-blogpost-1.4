 Task                  Command                                   
------------------------------------------
 Add a package         uv add fastapi      
 Remove a package      uv remove fastapi
 Sync environment      uv sync (matches .venv to your lockfile)
 Update all packages   uv lock --upgrade

For one-off scripts: 
uv run python my_script.py
or, uv run my_script.py <------------
Running Scripts Without Installation: uv run --with requests my_script.py

List available versions: uv python list
Install a specific version: uv python install 3.12
Pin project version (creates or updates .python-version): uv python pin 3.12

************************
after adding more packages to uv, we can still keep requirements.txt, to update: uv export > requirements.txt
or, professional: uv export --no-hashes --no-header --no-annotate --no-dev --output-file requirements.txt
